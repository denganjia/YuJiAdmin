# YuJi Admin 😛😛

使用`react`、`ant design v5`、`zustand`、`react-router`等打造的一体化、工程化、轻量化的 React 后台管理框架

# 快速上手 🚀

```shell
git clone https://github.com/denganjia/YuJiAdmin.git
yarn install #  or `npm install`
yarn dev # or `npm run dev`
```

# 特性 🔥🔥🔥
